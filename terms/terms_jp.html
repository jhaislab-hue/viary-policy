Viary 利用規約

施行日: 2025年10月25日

第1条（目的）
本規約は、Viary モバイルアプリの利用に関し、会社と会員との間の権利義務等を定めるものです。

第2条（定義）
「サービス」: Viary が提供する一切の機能／「会員」: 本規約に同意し登録した者／「コンテンツ」: 会員が作成した音声・メモ等のデータ／「有料サービス」: 対価を要するサブスクリプション等。

第3条（掲示・変更）
本規約はアプリ内に掲示します。不利益な変更は少なくとも30日前、それ以外は7日前までに通知します。

第4条（サービスの提供・変更）
録音・メモ管理・可視化・AI 機能等を提供します。内容や仕様は事前の告知のうえ変更・停止されることがあります。

第5条（会員の義務）
なりすまし、知的財産権侵害、リバースエンジニアリング・ハッキングその他違法または不当な行為を行ってはなりません。

第6条（コンテンツの権利・管理）
コンテンツの権利は会員に帰属します。コンテンツは端末内に保存され、当社サーバーには保存されません。AI 分析等の機能を利用する場合、必要最小限のデータがプライバシーポリシーに基づき外部 API に送信されることがあります。サービス自体に関する知的財産権は当社に帰属します。

第7条（有料サービス・決済）
料金・条件はアプリ内に掲示します。決済は各アプリマーケット（Apple/Google）の方式に従い、取消・返金は各ポリシーに従います。

第8条（解約・利用制限）
会員はいつでもアカウント削除により解約できます。会員が本規約に違反した場合、当社は通知のうえ利用を制限し、または解約できます。

第9条（免責）
不可抗力、会員の責に帰す事由、会員生成コンテンツの正確性、第三者サービス（Firebase/RevenueCat/Gemini API など）の障害等について、当社の故意または重過失がない限り責任を負いません。

第10条（準拠法・管轄）
本規約は大韓民国法に準拠し、紛争は所定の管轄裁判所に提起します。

附則
本規約は2025年10月25日から施行します。
